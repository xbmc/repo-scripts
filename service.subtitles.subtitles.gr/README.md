======================
Subtitles.gr addon for Kodi
======================

About
-----
Search and download subtitles from subtitles.gr, xsubs.tv and subztv.gr

Kodi Addon for http://www.subtitles.gr/, http://xsubs.tv/ and http://subztv.gr/
