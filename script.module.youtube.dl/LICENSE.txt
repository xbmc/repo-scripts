Several licenses apply to script.module.youtube.dl addon:

An GPL V2 license for the addon code to interface with yt-dlp.
The license is in the file: ./LICENSE-addon.txt

An unlicense for the yt-dlp, which this addon is a wrapper for. The
complete yt-dlp codebase was not required for this addon. As aresult,
 only the ./lib/yt_dlp directory was copied directly from the yt-dlp's
project. The license for yt-dlp is in the file:
./lib/yt_dlp/LICENSE. The version information is contained in
./lib/yt_dlp/update-version.py. The yt-dlp product originated
from https://github.com/yt-dlp/yt-dlp.
